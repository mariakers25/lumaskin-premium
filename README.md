# LumaSkin | Calming Repair Serum

A premium, conversion-focused landing page concept for LumaSkin Calming Repair Serum. This project is a portfolio piece showcasing a "Quiet Luxury" skincare aesthetic with a mobile-first, editorial layout.

## ✨ Features

- **Luxury Art Direction:** Minimalist cream/charcoal palette with editorial serif typography.
- **Micro-Animations:** Fluid content reveals using `IntersectionObserver`.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
- **Conversion Focused:** Clear visual hierarchy and campaign-matched CTA styling.
- **Vercel Ready:** Optimized for zero-config deployment on Vercel.

## 🚀 Getting Started

### Prerequisites

You only need [Node.js](https://nodejs.org/) installed on your machine.

### Installation

No heavy dependencies are required. Simply clone the repository and you are ready to go.

### Running Locally

To start a local development server:

```bash
npm run dev
```

The page will be available at `http://localhost:3000` (or the port specified in terminal).

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (CSS Variables) and Grid/Flexbox layouts.
- **JavaScript (ES6):** Vanilla JS for DOM interactions.
- **Fonts:** Cormorant Garamond (Serif) & DM Sans (Sans-serif) via Google Fonts.
